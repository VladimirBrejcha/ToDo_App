# ToDo_App
Simple todo app based on Swift 5 && Realm
![Frame 3](https://user-images.githubusercontent.com/44097057/57979386-8e392880-7a25-11e9-86ff-a7ed2f06403a.png)

### Features
- Categories
- Lists
- Realm
- Swift 5

### Libraries used:
- RealmSwift
- SwipeCellKit
- ChameleonFramework
